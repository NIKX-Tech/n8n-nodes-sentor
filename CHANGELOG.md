# Changelog

## [0.3.0] - 2026-01-06

### Added
- **Environment Selector**: Choose between Production and Development environments in the node settings.
- **Clustering**: New operation to cluster documents using BERTopic.
- **Topic Naming**: New operation to generate descriptive names for clusters using Google Gemini or Sentor API.
- **Google Gemini API Credential**: Support for providing a custom Google API key for topic naming.

### Changed
- **Documentation**: consolidated separate documentation files into `README.md`.
- **Error Handling**: Improved error reporting and type safety.
